## 영상 다양화 패턴 (쇼츠 시리즈)

같은 30초 hook 스타일 재활용 X. **4가지 패턴 로테이션:**

### 패턴 1: 사례 진단형 (가장 효과적)
"보증금 X / 월세 Y / 소득 Z, 자격 OK?" → 진단 → 결과
- 빠른 hook + 진단 결과 = 보는 사람이 자기 케이스 대입
- 비용 낮음 (15초로도 가능)

### 패턴 2: 데이터 인사이트형
"어제 N명 진단했는데 N% NO 받음. 이유는?" → 데이터 시각화 → 사이트 권유
- 권위 형성. "다른 사람들이 모이는 곳"
- 알고리즘 친화 (데이터 = 가치)

### 패턴 3: 비교형
"청년월세 vs 청년도약계좌 vs 행복주택 — 뭐가 더 받기 쉬워?"
- 폭넓은 검색 트래픽
- 사용자 의사결정 도움

### 패턴 4: 가이드/하우투형
"임대차계약서 확정일자 어디서 받아? 5분 가이드"
- SEO/검색 친화
- 사이트로 자연 연결 (서류 안내 강화)

각 패턴 1~2개씩 13일 동안. 같은 패턴 너무 자주 X.
