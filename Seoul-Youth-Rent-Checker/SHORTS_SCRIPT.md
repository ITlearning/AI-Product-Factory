# 쇼츠 제작 가이드 — 청년월세 체커

플랫폼: YouTube Shorts / Instagram Reels / TikTok 동시 게시.
포맷: 9:16 세로 / 30~45초 / 자막 필수 (음소거 시청 80%).

Tabber 시나리오 (5단계) 기반 BAB(Before-After-Bridge) 구조 + content-creator 인사이트.

## 첫 3초 HOOK이 80% 결정

**Bad:** "안녕하세요, 청년월세 자가진단 사이트 만들었습니다"
**Good:** "240만원 받을 수 있는데 이거 누가 봐요?" + 즉시 PDF 화면

쇼츠는 첫 3초에 스크롤 결정. **질문 던지기 + 시각 충격** = 멈춤.

대안 hook 후보:
- "청년월세 240만원, 너도 받을 수 있어?" (Landing 헤드라인 그대로)
- "PDF 11장 다 읽어야 됨. 누가 읽을까?" (좌절 공감)
- "이거 모르면 240만원 날리는 거" (FOMO)

## 촬영 팁

### Mac 화면 녹화
- **QuickTime Player** > File > New Screen Recording > 영역 선택 > 녹화
- 또는 **Cmd+Shift+5** > "선택 영역 기록"
- 1080×1920 (9:16) 또는 1920×1080 (16:9)으로 녹화 후 편집에서 crop

### iPhone Safari 화면 녹화
- **제어 센터 > 화면 기록** (Vercel preview URL 열어둔 채)
- Safari로 사이트 접속해서 사용하는 모습 — 더 친근

### 톤 권장
- **Tabber 본인 얼굴 X** (선택. 등장하면 더 친근하지만 부담 시 PASS)
- **컷 빠르게** (1~2초마다 컷 전환). 한 컷 4초 넘으면 시청자 이탈
- **자막은 큰 글씨, 흰색 + 검은 외곽선** (모바일에서 가독성)

## 자막 카피 후보 (각 컷별)

### 컷 1: HOOK (0~3초)
- A안: "240만원 받을 수 있는데 이거 누가 봐요?" (질문 + 공감)
- B안: "월세 240만원 지원금 신청하려면..." (cliffhanger)
- C안: "이 PDF 11장 다 읽었어요?" (도전적)

→ 추천 A. 질문이 가장 강함.

### 컷 2: AGITATE (3~8초)
- "조건이 16개"
- "군복무 보정? 환산식? 4구간?"
- "PDF가 11장이야"

→ 두 줄 정도. 빠르게 넘어감.

### 컷 3: TRANSITION (8~12초)
- "그래서 만들었음" (담담)
- 또는 "직접 만들었음" (자랑 톤)

→ 추천 첫번째. 겸손해야 친근.

### 컷 4: DEMO (12~25초)
한 동작당 한 자막:
- 생년월일 입력 → "5분이면 끝"
- 가구형태 선택 → "1인이든 신혼이든"
- 보증금/월세 → "공인인증서 X"
- 소득/재산 → "익명, 결과는 본인만"
- 결과 화면 → "진짜 4구간까지 알려줌"

### 컷 5: PROOF (25~28초)
- "1구간 35% 추첨"
- "필요 서류 4종 발급처까지"

### 컷 6: CTA (28~30초)
- "나도 해보고 싶으면 댓글 OK" (가장 약한 CTA, 부담 X)
- "링크는 프로필" (인스타 룰 — 캡션에 "링크는 프로필" 명시)
- "댓글에 OK 달면 DM 보내드림"

→ 추천: **"나도 해보고 싶으면 댓글에 'OK'"** + 인스타 캡션에 링크.

## BGM/효과음

### 추천 (저작권 free)
- **TikTok 라이브러리:** "trending" 카테고리에서 차분한 lo-fi
- **YouTube 오디오 라이브러리:** https://studio.youtube.com → Audio Library
- **Pixabay Music:** https://pixabay.com/music/ (무료, 광고 X)

### 분위기
- 컷 1~2 (HOOK + AGITATE): 약간 unsettled, 빠른 비트
- 컷 3 (TRANSITION): 정적 (단발 sfx)
- 컷 4~5 (DEMO + PROOF): 차분한 lo-fi (서비스 신뢰감)
- 컷 6 (CTA): 페이드아웃

### 효과음 (sfx)
- 컷 1: 키보드 타이핑 (chk chk chk)
- 컷 2: "어... 어..." 좌절 (직접 녹음 또는 라이브러리)
- 컷 3: 단발 하이햇 또는 "ding"
- 컷 5: 결과 등장 시 "성공" sfx (가벼움)

## 게시 일정

### 1차 (5/5 밤): "공식 출시" 쇼츠
시나리오: 위 30초 그대로. 신청 시작 12시간 전 게시.

### 2차 (5/9 즈음): "이런 사람 자격 OK" 쇼츠
사례별 짧은 영상:
- "보증금 5천 + 월세 50, 이 분 자격 OK?" → 진단 → "1구간 35%"
- "임대인이 부모면?" → "FAIL"
- 데이터 공유 = 콘텐츠 확장

### 3차 (5/15 즈음): "마감 임박" 쇼츠
"D-4. 5분만 투자해" — 카운트다운 영상.

## 측정

쇼츠 첫 24시간 지표:
- View > 1,000: 정상
- View > 10,000: 알고리즘 푸시
- View > 100,000: 폭발

링크 클릭률 (CTR):
- YouTube 설명란: 0.5~1% 정상
- Instagram 캡션 "링크 프로필": 1~3% 정상
- TikTok 프로필 링크: 0.5~1% 정상
