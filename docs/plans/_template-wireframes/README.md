# Wireframe Template

이 디렉터리는 PRD에 첨부하는 **HTML 와이어프레임의 시드**다. 새 PRD에 와이어프레임을 붙일 때 여기 파일을 복사해서 시작점으로 쓴다.

## 사용 방법

1. 새 PRD를 시작할 때 `docs/plans/<date>-<topic>-wireframes/` 디렉터리를 만든다.
2. 이 폴더의 `recommend-home.html` / `recommend-empty.html`을 복사해 화면 이름에 맞게 이름을 바꾼다.
3. `<header>` 섹션의 화면 이름·진입 경로·핵심 액션 메타정보를 자기 PRD에 맞게 수정한다.
4. `<main>` 안 박스 레이아웃을 자기 화면 구조에 맞게 다시 짠다.
5. 브라우저에서 단독으로 열어 확인한다 (외부 의존성 없이 동작해야 한다).

## 디자인 원칙

- **목적**: 픽셀 충실도가 아니라 **정보 구조와 사용자 동선**을 보여준다. 실제 색상·폰트·간격이 픽셀 단위로 맞을 필요 없다.
- **스타일**: `<style>` 블록을 파일 안에 인라인으로 둔다. 외부 CSS·JS·이미지 없이 단독 파일로 동작.
- **헤더 메타**: 모든 와이어프레임 상단에 화면 이름 / 진입 경로 / 핵심 액션을 한눈에 보이게 둔다.
- **상태**: 같은 화면의 다른 상태(빈 상태, 에러, 로딩)는 별도 파일로 둔다 — 한 파일에 여러 상태를 욱여넣지 않는다.
